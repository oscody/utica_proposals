# Minnesota Security Company License

## 1. License Name

- Protective Agent License / Protective Agent Company License
- Regulated by Minnesota Board of Private Detective and Protective Agent Services

## 2. Core Requirement

- Minnesota requires a state protective-agent license to operate a contract security business.

## 3. Owner / Qualification Requirements

- Minnesota licenses the individual qualifying protective agent and the company
- Application includes background and statutory qualifications review

## 4. Business Requirements

- File through Minnesota DPS licensing
- Use the Minnesota licensing forms for the company and for each qualifying individual

## 5. Insurance / Bond

- Confirm the current bond or insurance requirement in the Minnesota application materials before filing

## 6. Fees & Validity

- Company license fee should be confirmed from Minnesota DPS current materials when filing

## 7. Employee Guard Requirements

- Minnesota separately licenses protective-agent employees

## 8. Simple Summary

1. Form the company
2. License the qualifying protective agent
3. Apply for the company license
4. Confirm current insurance and fee items in the Minnesota packet
5. License employee protective agents
