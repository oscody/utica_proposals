# Ohio Security Company License

## 1. License Name

- Class C Security Guard Provider License
- Regulated by Ohio Department of Public Safety, Private Investigator Security Guard Services

## 2. Core Requirement

- A company offering security guard services in Ohio needs a Class C license.
- Ohio requires a qualifying agent and is more compliance-driven than a simple registration state.

## 3. Owner / Qualifier Requirements

- For at least 2 continuous years immediately before application, the applicant or qualifying agent must have qualifying security / investigative / related experience described in Ohio law
- Must pass the state examination
- Must not be legally disqualified under the statute
- A corporation can qualify through an officer or qualifying agent

## 4. Business Requirements

- File the provider application with Ohio DPS
- Use a qualifying agent if applying as a company
- Apply separately for branch offices

## 5. Insurance / Bond

- Comprehensive general liability insurance or equivalent guarantee required
- Minimum coverage:
  - $100,000 for each person
  - $300,000 for each occurrence for bodily injury
  - $100,000 for property damage

## 6. Fees & Validity

- Exam application: $25
- Retained applicant fingerprint database fee: $5
- Initial license issuance: $375
- Branch license: $100
- Annual renewal: $275

## 7. Employee Guard Requirements

- Employees must be registered with the state
- Registration card issuance: $40
- Annual registration renewal: $25
- Firearm bearer notation: $15 if armed

## 8. Simple Summary

1. Form the company
2. Make sure your qualifying agent meets Ohio’s 2-year experience rule
3. Pass the required exam
4. Get the required insurance
5. Apply for the Class C provider license and register each employee
