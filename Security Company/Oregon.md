# Oregon Security Company License

## 1. License Name

- Private Security Entity License
- Regulated by Oregon Department of Public Safety Standards and Training

## 2. Core Requirement

- Oregon requires a state private security entity license for a contract security business.

## 3. Owner / Qualification Requirements

- Oregon licenses the business entity and separately certifies individual private security professionals

## 4. Business Requirements

- Apply through the DPSST private security program
- Business and controlling-person information are part of the filing

## 5. Insurance / Bond

- Confirm Oregon’s current insurance or bond filing requirement in the DPSST application materials before filing

## 6. Fees & Validity

- Oregon uses state-set licensing fees
- Confirm the current exact entity fee with DPSST before filing

## 7. Employee Guard Requirements

- Oregon private security professionals are certified separately

## 8. Simple Summary

1. Form the company
2. Apply for the Oregon entity license
3. Confirm current fee and insurance items with DPSST
4. Certify individual private security professionals

## 9. Sources

- Oregon DPSST private security page: https://www.oregon.gov/dpsst/PS/Pages/default.aspx
