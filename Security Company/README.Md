# Security Company Licensing Directory

This folder contains concise state-by-state notes on how to get licensed to operate a private security company.

Each file is for quickly finding:

- the agency license name
- the state regulator
- owner or qualifying-agent requirements
- insurance or bond requirements
- fees and renewal terms
- employee guard licensing rules

Use these files as a quick internal reference for comparing states and planning expansion.

They are not legal advice and should be checked against the official state source before filing.

`Florida.md` is the original example and stays unchanged.
