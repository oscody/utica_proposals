# Montana Security Company License

## 1. License Name

- Private Security Company License
- Regulated by Montana Board of Private Security

## 2. Core Requirement

- Montana requires a state license for a private security company.

## 3. Owner / Qualification Requirements

- Montana regulates companies and individual private security guards through the same board
- Background review is part of the licensing process

## 4. Business Requirements

- Apply through Montana’s licensing portal
- Use the exact Montana private security company license type

## 5. Insurance / Bond

- Confirm current bond or insurance requirements with the Montana board packet before filing

## 6. Fees & Validity

- Montana uses online licensing and board-set fees
- Confirm the current exact company fee in the portal

## 7. Employee Guard Requirements

- Guards need Montana private security guard licensing

## 8. Simple Summary

1. Form the company
2. Apply for the Montana private security company license
3. Confirm current fee and insurance items in the portal
4. License all guards separately

## 9. Sources

- Montana private security board page: https://boards.bsd.dli.mt.gov/private-security/
