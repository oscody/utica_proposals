# Arkansas Security Company License

## 1. License Name

- Security Services Contractor / Private Security Agency License
- Regulated by Arkansas State Police Regulatory Services Division

## 2. Core Requirement

- Arkansas licenses private security agencies at the state level.
- A company cannot legally operate without the Arkansas State Police license.

## 3. Owner / Qualification Requirements

- Applicant must be at least 18
- Must meet Arkansas background and character standards
- Arkansas law applies additional requirements depending on the license category and role

## 4. Business Requirements

- Apply through Arkansas State Police Regulatory Services
- Arkansas publishes statutes, rules, and agency forms through the ASP licensing program

## 5. Insurance / Bond

- Bond and insurance requirements should be confirmed in the current ASP packet for the exact license type

## 6. Fees & Validity

- Arkansas uses a formal state licensing process
- Verify the current company fee directly with Arkansas State Police before filing

## 7. Employee Guard Requirements

- Arkansas separately credentials guards and commissioned security officers
- Armed personnel need the appropriate commissioned status

## 8. Simple Summary

1. Form the company
2. Choose the correct Arkansas security license category
3. Meet ASP background and business requirements
4. Confirm bond and fee items with the current packet
5. Credential employees before deployment
