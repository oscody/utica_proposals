# Illinois Security Company License

## 1. License Name

- Private Security Contractor Agency License
- Regulated by Illinois Department of Financial and Professional Regulation

## 2. Core Requirement

- Illinois requires a state agency license for a contract security company.
- The agency must maintain a licensed licensee-in-charge.

## 3. Owner / Qualification Requirements

- Agency must have a full-time Illinois licensed private security contractor as licensee-in-charge
- This is a continuing requirement
- Corporate officers and directors are reviewed for character standards

## 4. Business Requirements

- Apply through IDFPR CORE
- Branch offices are separately licensed
- If out of state, records still must be maintained as Illinois rules require

## 5. Insurance / Bond

- Confirm current insurance and financial responsibility requirements with IDFPR application materials

## 6. Fees & Validity

- Illinois has agency, branch, and individual license fees through IDFPR
- Confirm the current exact fee schedule in CORE when filing

## 7. Employee Guard Requirements

- Employees generally work under PERC and firearm-control-card structures, depending on role
- Armed personnel need the appropriate firearm credentials

## 8. Simple Summary

1. Form the company
2. Make sure you have a full-time Illinois licensee-in-charge
3. Apply for the agency license through IDFPR
4. Handle branch licensing if needed
5. Make sure employees hold the right Illinois credentials
