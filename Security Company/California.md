# California Security Company License

## 1. License Name

- Private Patrol Operator License
- Regulated by the California Bureau of Security and Investigative Services

## 2. Core Requirement

- California requires a PPO license to operate a contract security company.
- This is one of the main agency-license structures in the country.

## 3. Owner / Qualification Requirements

- A qualified manager is required
- California separately licenses the Private Patrol Operator Qualified Manager
- Principals and corporate officers are part of the BSIS filing process

## 4. Business Requirements

- Apply through BSIS / BreEZe
- Branch offices require separate branch licensing
- The company must stay compliant with BSIS reporting and recordkeeping rules

## 5. Insurance / Bond

- California requires PPO liability insurance
- Use the BSIS PPO insurance requirements with the application packet

## 6. Fees & Validity

- PPO initial application and exam: $605
- PPO initial license: $847
- Total initial fee: $1,452
- PPO renewal: $990
- Branch office initial: $275
- Branch office renewal: $165
- Qualified manager initial application and exam: $385

## 7. Employee Guard Requirements

- Guards need California Security Guard registration
- Initial guard application: $60
- Guard renewal: $44
- Armed work requires a firearms permit

## 8. Simple Summary

1. Form the company
2. Get the PPO qualified manager lined up
3. Apply through BSIS
4. Meet California insurance requirements
5. Register each guard and firearms permit holder
